# Summary

You have to open this print in the customizer to fit it to your router and material. 

It is used to create floating tenon joints between two pieces of wood. Look at the following video for a description of all parameters:

https://youtu.be/n9KUrZueMNI

The default parameters work with a 17mm guide bushing and festool domino tenons.
